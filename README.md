# 视频脚本字数统计与时间预估工具

## 简介
这是一个专门为儿童教学视频脚本设计的字数统计和时间预估工具。它能够自动统计脚本各部分的字数（排除括号内的后期制作提示），并基于儿童教学语速（220字/分钟）预估每个部分的口播时间。

## 功能特点
- ✅ 自动识别脚本的四个固定部分（引入、知识点讲解、综合练习、总结）
- ✅ 排除所有类型括号内的内容进行字数统计
- ✅ 基于220字/分钟的儿童教学语速计算时间
- ✅ 生成累积时间轴（00:00-XX:XX格式）
- ✅ 在原文档标题后自动添加字数和时间标注

## 🌟 推荐：网页版（无需安装任何软件）

**这是最简单的使用方式！**

### 使用步骤
1. 双击打开 `video_script_counter.html` 文件
2. 拖拽或点击上传你的 .docx 文件
3. 点击"开始处理"按钮
4. 自动下载带标注的文件

## Python命令行版本（需要Python环境）

### 安装

```bash
pip install -r requirements.txt
```

### 使用方法

```bash
python video_script_counter.py <输入文件.docx>
```

输出文件将保存为：`<输入文件>_带标注.docx`

## 示例

输入文档：
```
第一部分：引入
这是一段引入文字（展示动画效果）...

第二部分：知识点讲解
知识点1
详细讲解内容...
```

输出文档：
```
第一部分：引入（约150字，00:00-00:41）
这是一段引入文字（展示动画效果）...

第二部分：知识点讲解（约800字，00:41-04:00）
知识点1
详细讲解内容...
```

## 技术规格
- 支持格式：.docx
- 语速：220字/分钟（可在代码中配置）
- 支持括号类型：()、（）、[]、【】、「」、『』、{}、｛｝
- 字数统计：包含中文、英文、数字、标点符号
- 时间计算：四舍五入到整秒
